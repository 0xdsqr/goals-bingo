# Goals Bingo

Track your yearly goals with a bingo board. Add goals, check them off, get a BINGO.

## Getting Started

```bash
# Enter dev shell
nix develop

# Install deps
bun install

# Start Convex backend
cd apps/bingo
bun run dev:convex

# In another terminal, start the app
bun run dev
```
